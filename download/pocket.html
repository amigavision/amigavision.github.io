Download AmigaVision for Analogue Pocket

The AmigaVision team does not distribute software, we only maintain an open source script that builds a setup if you supply your own demos and games.

We can point you to the search page of the software preservation section of archive.org, but note that anything available there is not built by—nor distributed by—the team, and the team has no responsibility for these files.

Installation Steps

  1. Install the OpenFPGA Amiga core using your preferred Pocket updater, or manually add the latest Amiga Pocket Core.
  2. Download the AmigaVision archive from archive.org and extract it.
  3. Copy the files in games/Amiga to /Assets/amiga/common on your Pocket SD card.
  4. If you already have a save file, make sure it is named AmigaVision-Saves.hdf (not MegaAGS-Saves.hdf).
  5. Start the Amiga core and enjoy.

Pocket Controls

  • Select shows the on‑screen keyboard (press DEL to quit a game).
  • Start toggles mouse emulation mode.
  • L/R triggers are the left/right mouse buttons.
  • A selects, B goes back.
