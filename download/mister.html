Download AmigaVision for MiSTer

The AmigaVision team does not distribute software, we only maintain an open source script that builds a setup if you supply your own demos and games.

We can point you to the search page of the software preservation section of archive.org, but note that anything available there is not built by—nor distributed by—the team, and the team has no responsibility for these files.

Installation Steps

  1. Download AmigaVision
    and extract the archive on your computer.
  2. Copy the following folders to the root of your MiSTer SD card:
_Computer
_Console
config
Filters
games
Presets
Shadow_Masks
  1. Add the recommended Amiga core settings to the bottom of your MiSTer.ini:
[Amiga
+Amiga500
+Amiga500HD
+Amiga600HD
+AmigaCD32]
video_mode_ntsc=8 ; These two use the recommended setting of 1080p60 and
video_mode_pal=9  ; 1080p50, adjust if you want a different resolution
vscale_mode=0
vsync_adjust=1 ; You can set this to 2 if your display can handle it
custom_aspect_ratio_1=40:27
bootscreen=0
  1. Reboot MiSTer.
  2. Launch Amiga from the Computer section (this is the main AmigaVision setup).
