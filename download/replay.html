AmigaVision for RePlayOS

The AmigaVision team does not distribute Amiga software, we only maintain an open source script that builds a setup if you supply your own demos and games.

We can point you to the search page of the software preservation section of archive.org, but note that anything available there is not built by—nor distributed by—the team, and the team has no responsibility for these files.

Installation

  1. ⬇️ Download AmigaVision for Emulators
    and extract the archive on your computer.
  2. Copy the file Amiberry/Harddrives/AmigaVision.hdf (it uses the same hard disk file as the emulators) to roms/commodore_ami.
  3. Make sure you have the latest RePlayOS update installed using the online updater, minimum version required is 1.6.4.
  4. For optimal Amiga performance and proper 50hz/60hz switching — important for the Amiga! — set these options in the RePlayOS settings:
    • Video → Video Mode = LCD Native@60/NRR
    • Video → LCD Type = TV/PC VRR
    • Video → Integer Scale → Vertical
    • Video → Scanlines: Light
  5. You can launch straight into by AmigaVision by selecting it under the Commodore Amiga entry in the menu. If you want to boot straight into AmigaVision on boot, you can optionally put this file in roms/_autostart — click the download icon next to where it says Raw on the GitHub page to get the file.
  6. Once you have started AmigaVision, for optimal Amiga scaling — bring up the RePlayOS menu, and set these options in the RePlayOS system-specific settings for Amiga:
    • Video → Crop: Automatic
    • Video → Crop Mode: Vertical
  7. Enjoy the ultimate Amiga games & demo scene collection!

Notes

  • Default resolution is 1080p, but you can change this in RePlayOS settings — press the Windows key to bring up the menu. Consult the RePlayOS documentation for full details.
  • Currently, the emulator used in this setup will not run the “060 Demos”, but this will be addressed in the future.
