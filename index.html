User-Friendly Launcher

Experience the best of the Amiga system with our sophisticated and performant game and demo launcher with pixel-perfect high-resolution screenshots, and metadata like release date, developer, publisher, etc.

Entirely controllable using gamepads, joysticks or via keyboard. This lets you quickly and easily try the best of what the system has to offer.

Well-Tested & Curated

Our carefully curated and well-tested settings for games and demos mean no duplication of AGA and ECS versions, with lots of genre and top lists to help you navigate the massive amount of Amiga games available. We take special care to run every game at the correct aspect ratio and CPU speed.

Optimized for 4K & HD

Explore more than 3000 games and almost 600 demos with optimal settings, with hand-tuned, per-game 5× Integer Overscale, all of which make the best use of modern high-resolution displays without leaving large parts of the screen blank—and without suffering from scaling artifacts.

Additionally, Dynamic Pixel Aspect Ratio ensures that the Amiga’s original PAL ratio of 16:15 and NTSC ratio of 5:6 is preserved correctly and automatically for every game.

Best of the Demo Scene

Discover the legendary Amiga demo scene with our included key productions, including disk magazines sorted chronologically. Our launcher is the perfect companion to explore the demo scene’s UNESCO-nominated cultural heritage.

Authentic Look & Feel

Hand-tuned scan lines and shadow mask settings are included to get you close to that original CRT look if you are using it on a modern display. The analog output will of course be unaffected. On MiSTer & Raspberry Pi, you can output HDMI to your viewers and simultaneously play on analog CRTs at the same time, perfect for streaming.

Shared Network Drives

We make it easy to transfer files to and from the Amiga via Wi-Fi or wired networks—or directly using the SD card, with shared network file system support for MiSTer.

Robust File System

Our setup uses Professional File System as its file system to avoid accidental corruption on write operations, which the standard file system is very prone to. This even works on Workbench 1.3! Our minimalist Workbench setup also includes support for your own custom set of configurations, games, applications, and files.

High Resolution Displays

With the included RTG support, you can run Workbench in higher resolutions like 1920×1080 and in 16:9 aspect ratio on modern displays.

Cycle-Accurate Amiga 500

Includes a dedicated setup to closely mimic a stock Amiga 500 with memory expansion for maximum compatibility with demo scene productions and any troublesome games that rely on trackloaders, cycle accuracy and exact hardware timings.
