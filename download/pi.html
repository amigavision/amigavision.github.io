Download AmigaVision for Raspberry Pi

The AmigaVision team does not distribute software, we only maintain an open source script that builds a setup if you supply your own demos and games.

We can point you to the search page of the software preservation section of archive.org, but note that anything available there is not built by—nor distributed by—the team, and the team has no responsibility for these files.

Installation Steps (RePlayOS)

  1. Download and install RePlayOS on your Raspberry Pi.
  2. Boot RePlayOS once to create the replay exFAT partition.
  3. Download the AmigaVision archive from archive.org and extract it.
  4. Place the required Kickstart ROMs in the bios directory, following the RePlayOS BIOS naming guide.
  5. Copy games/Amiga/AmigaVision.hdf to roms/commodore_ami on the RePlayOS SD card.
  6. Return the SD card to the Raspberry Pi and launch AmigaVision under Commodore Amiga.

Optional: Auto‑Start AmigaVision

  1. Add AmigaVision.hdf to favorites inside RePlayOS.
  2. Copy the generated .fav file into the _autostart folder.
