Download AmigaVision for Raspberry Pi

The AmigaVision team does not distribute Amiga software, we only maintain an open source script that builds a setup if you supply your own demos and games.

We can point you to the search page of the software preservation section of archive.org, but note that anything available there is not built by—nor distributed by—the team, and the team has no responsibility for these files.

Installation Steps

AmigaVision runs well on any Raspberry Pi 5, 500, 500+, or CM5 with 1GB of RAM — or more, but 1GB is sufficient. Raspberry Pi 4 is also supported, but too slow for the most accurate emulation.

  1. ⬇️ Download AmigaVision for Raspberry Pi
  2. Install a disk imaging tool — we recommend Balena Etcher or Raspberry Pi Imager. If you use Win32 Disk Imager on Windows, you'll have to unpack the archive using 7zip or similar before flashing it to the card since it has no support for .xz files.
  3. Write the image to your SD/CF card using your imaging tool.
  4. Insert the SD/CF card into your Raspberry Pi, turn it on.
  5. Enjoy the ultimate Amiga games & demo scene collection!

Notes

  • Default resolution is 1080p, but you can change this in RePlayOS settings — press the Home key or the Windows key to bring up the menu. Consult the RePlayOS documentation for full details.
  • Default scaling is currently 3× instead of 4×, this will be fixed in the next release of RePlayOS.
  • The emulator is currently not running in cycle-accurate mode, this will be fixed in the next release of RePlayOS.
