AmigaVision for Analogue Pocket

The AmigaVision team does not distribute Amiga software, we only maintain an open source script that builds a setup if you supply your own demos and games.

We can point you to the search page of the software preservation section of archive.org, but note that anything available there is not built by—nor distributed by—the team, and the team has no responsibility for these files.

Installation Steps

  1. ⬇️ Download the Pocket Amiga Core
    and install it, or install it using your preferred Pocket updater — we recommend Pupdate.
  2. ⬇️ Download AmigaVision for Emulators & Analogue Pocket and extract the archive on your computer.
  3. Copy the files inside the Amiberry/Harddrives folder (it uses the same hard disk files as the emulators) to /Assets/amiga/common on your Pocket SD card.
  4. If you already have a save file, make sure it is named AmigaVision-Saves.hdf (not MegaAGS-Saves.hdf).
  5. Enjoy the ultimate Amiga games & demo scene collection!

Pocket Controls

  • Select shows the on‑screen keyboard (press DEL to quit a game).
  • Start toggles mouse emulation mode.
  • L and R triggers are the left/right mouse buttons.
  • A selects an entry in the launcher, B goes back to the parent category.
  • Many games are updated to support two-button controls, but some require remapping if you prefer button-to-jump instead of up-to-jump (which makes sense on a joystick, but is pretty terrible on a gamepad). Button remapping is done just like in any other Pocket core.
